Wanderly is a smart travel planning application that transforms destinations and budgets into actionable, personalized itineraries using AI orchestration.

## 🚀 Core Features
* **AI-Powered Generation:** Leverages LLMs to sequence activities based on user-defined interests and budget.
* **Freemium Flow:** "Start Planning Free" allows users to generate a **Day 1 Preview** instantly without an account.
* **Intelligent Paywall:** Advanced features (Day 2-5, Saving, Exporting) are locked behind a contextual login modal.
* **Secure Auth:** Streamlined single-step login restricted to **Gmail accounts**.

## 🛠️ Technologies Used
This project is built with a modern, high-performance stack:
* **Framework:** [React](https://reactjs.org/) with [Vite](https://vitejs.dev/) for ultra-fast development.
* **Language:** [TypeScript](https://www.typescriptlang.org/) for type-safe, reliable code.
* **UI Components:** [shadcn/ui](https://ui.shadcn.com/) for beautiful, accessible components.
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) for a custom, responsive design.
* **Platform:** Developed and deployed via [Lovable](https://lovable.app/).

## 🛠️ Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
